## About the Tailscale Operator

The Tailscale Kubernetes Operator allows you to manage Tailscale resources using Kubernetes-native APIs. It enables
secure connectivity to your Kubernetes cluster and services via a tailnet, including:

- Exposing Kubernetes services to your tailnet.
- Accessing your tailnet from within the cluster.
- Managing Tailscale authentication and configuration via Custom Resource Definitions (CRDs).

For more details on the Tailscale operator, visit the
[official documentation](https://tailscale.com/kb/1236/kubernetes-operator).

## About Docker Hardened Images

Docker Hardened Images are built to meet the highest security and compliance standards. They provide a trusted
foundation for containerized workloads by incorporating security best practices from the start.

### Why use Docker Hardened Images?

These images are published with zero-known CVEs, include signed provenance, and come with a complete Software Bill of
Materials (SBOM) and VEX metadata. They're designed to secure your software supply chain while fitting seamlessly into
existing Docker workflows.
