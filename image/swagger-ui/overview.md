# Swagger UI

`swagger-ui` allows anyone — be it your development team or your end consumers — to visualize and interact with the API’s resources without having any of the writeback logic in place. It’s automatically generated from your OpenAPI (formerly known as Swagger) Specification, with the visual documentation making it easy for back-end implementation and client-side consumption.

## Features

- **Interactive Documentation**: Test and execute API calls directly from the browser interface.
- **Support for OpenAPI Standard**: Fully supports OpenAPI 2.0, 3.0, and 3.1 specifications.
- **Customizable & Themed**: Configurable layouts, security controls, and endpoint grouping behaviors.
